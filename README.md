БЛОК 1. Реакция интерфейса на действия пользователя

1.  Название: "Отсутствие валидации при вводе значения"
    Шаги воспроизведения:
      1) Начать вводить "user"
      2) Не нажимать кнопку "Проверить"
     Ожидаемый результат: Сообщение об ошибке не выводится, alert не вызывается


БЛОК 2. Позитивные сценарии

2.  Название: "Ввод стандартного email-адреса"
    Шаги воспроизведения:
      1) Ввести в поле ввода "user@academy.ru"
      2) Нажать кнопку "Проверить"
    Ожидаемый результат: выводится alert с текстом "Прекрасный email-адрес"

3.  Название: "Ввод email с допустимыми символами"
    Шаги воспроизведения:
      1) Ввести в поле ввода "user123!#$%&‘*+—/=?^_`{|}~@academy.ru"
      2) Нажать кнопку "Проверить"
    Ожидаемый результат: выводится alert с текстом "Прекрасный email-адрес"

4.  Название: "Ввод email, содержащего цифры в имени и домене"
    Шаги воспроизведения:
      1) Ввести в поле ввода "user123@academy456.ru"
      2) Нажать кнопку "Проверить"
    Ожидаемый результат: выводится alert с текстом "Прекрасный email-адрес"


БЛОК 3. Негативные сценарии

5.  Название: "Ввод email с недопустимыми значениями"
    Шаги воспроизведения:
      1) Ввести в поле ввода "пользователь@академия"
      2) Нажать кнопку "Проверить"
    Ожидаемый результат: выводится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email"

6.  Название: "Ввод email без знака "@" "
    Шаги воспроизведения:
      1) Ввести в поле ввода "useracademy.ru"
      2) Нажать кнопку "Проверить"
    Ожидаемый результат: выводится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email"

7.  Название: "Ввод нескольких "@" "
    Шаги воспроизведения:
      1) Ввести в поле ввода "user@@@academy.ru"
      2) Нажать кнопку "Проверить"
    Ожидаемый результат: выводится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email"

8.  Название: "Ввод email без домена"
    Шаги воспроизведения:
      1) Ввести в поле ввода "user@"
      2) Нажать кнопку "Проверить"
    Ожидаемый результат: выводится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email"

9.  Название: "Ввод email с пробелами"
    Шаги воспроизведения:
      1) Ввести в поле ввода "use r@academy.ru"
      2) Нажать кнопку "Проверить"
    Ожидаемый результат: выводится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email"

10.  Название: "Ввод пустого значения"
